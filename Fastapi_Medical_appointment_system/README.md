**🏥 Medical Appointment System (FastAPI)**

**Project Overview:**
This is a backend system built using FastAPI that simulates a real-world medical appointment booking application.It allows users to manage doctors, schedule appointments, and track consultation workflows efficiently.

**Features:**
1.RESTful APIs using FastAPI
2.Pydantic-based request validation
3.Doctor management (add, update, delete)
4.Appointment booking system
5.Multi-step workflow (Schedule → Confirm → Cancel → Complete)
6.Fee calculation (appointment type + senior citizen discount)
7.Doctor filtering and search
8.Sorting and pagination
9.Combined browsing (search + sort + pagination)
10.Swagger UI for API testing

**Tech Stack:**
1.Python
2.FastAPI
3.Uvicorn
